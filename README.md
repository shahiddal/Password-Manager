# 🔐 Password Manager

A modern Password Manager built with React.js and Tailwind CSS that allows users to securely manage website credentials in a clean and responsive interface.

## 🚀 Features

- Add new passwords
- Save website URL, username, and password
- View saved credentials in a table format
- Copy credentials with one click
- Delete saved passwords
- Responsive UI
- Modern design using Tailwind CSS
- React Context API for state management

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- Context API

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/shahiddal/Password-Manager.git
```

2. Navigate to project directory

```bash
cd Password-Manager
```

3. Install dependencies

```bash
npm install
```

4. Start development server

```bash
npm run dev
```
